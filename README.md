# BioGen Sertão

Sistema inteligente de apoio à gestão genética e reprodutiva para bovinos, ovinos e caprinos.

Desenvolvido para o Hackathon Expoagro Crateús 2026.

## Objetivo

O projeto busca auxiliar produtores rurais e técnicos na tomada de decisão reprodutiva através de um motor de inteligência adaptativa local, permitindo análises genéticas, controle de rebanho, histórico de cruzamentos e rastreabilidade.

## Funcionalidades

- Cadastro de animais
- Gestão de currais e lotes
- Histórico reprodutivo
- Simulação de cruzamentos genéticos
- Histórico de análises
- Assistente inteligente local
- Relatórios exportáveis
- Upload e análise de imagens
- Suporte mobile
- Funcionamento offline (PWA)

## Tecnologias

- React
- Vite
- LocalStorage
- PWA
- CSS Responsivo
